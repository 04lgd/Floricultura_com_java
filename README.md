# Sistema de Gerenciamento de Floricultura

Este é o meu primeiro projeto autoral, que irei desenvolver sozinha usando os conhecimentos adquiridos através da playlist gratuita **Maratona Java Virado no Jiraya**, no canal DevDojo, do Willian Suane, disponível no YouTube.

Atualmente (23/03/2026), sou estudante do curso de Sistemas de Informação, da Universidade Federal de Uberlândia, e irei iniciar o segundo semestre em um mês.

## Objetivo

O projeto visa gerenciar o estoque de uma floricultura, permitindo o cadastro de produtos, controle de quantidades e categorização inteligente.

## Conceitos aplicados

Como essa é minha primeira aventura com Java, o projeto evoluirá conforme avanço nas aulas da playlist. Inicialmente, serão aplicados alguns conceitos que já tive contato, tais como:

- **Orientação a Objetos**
- **Contrutores**
- **Herança**
- **Enums**
- **Associação entre classes**

## Tecnologias usadas

- Java 25
- Intellij IDEA
- Git/GitHub
